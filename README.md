# Worlds of Enchant

A fantasy adventure game built in C++ using Raylib.

## Build Instructions
- Requires CMake and a C++ compiler
- Run `cmake -S . -B build`
- Run `cmake --build build`

## Controls
- WASD to move
- Space to interact

## Project Structure
- src/ — game code
- assets/ — textures, fonts, audio
- saves/ — save files
